# X (Twitter) 記事告知テンプレート

## 基本フォーマット

```
[タイトル]

[1行で一次情報のポイント — 自分の体験・数字を入れる]

#ClaudeCode #Zenn
https://zenn.dev/shimo4228/articles/[SLUG]
```

### ポイント

- **一次情報を1行で**: 「やってみた結果」「具体的な数字」が刺さる
- **ハッシュタグ**: `#ClaudeCode` は必須。他は記事内容に合わせて1〜2個追加
- **画像添付**: スクリーンショットか図解を1枚添付するとインプレッション大幅増
- **投稿時間**: 平日 8:00〜10:00（エンジニアの朝の情報収集タイム）
